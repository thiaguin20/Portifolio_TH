## Sobre o projeto

Este portfolio foi desenvolvido para funcionar como vitrine dos meus principais trabalhos. A home usa uma composicao espacial/holografica com navegacao por secoes, enquanto a area de projetos prioriza videos curtos, descricao objetiva, tecnologias usadas, link do repositorio e link online quando o projeto esta publicado.

## Projetos em destaque

- **Cliente Mobile**: CRM mobile offline desenvolvido com Flutter, Dart, SQLite e sqflite.
- **Estoque Web**: sistema ASP.NET MVC para controle de estoque, publicado no Azure App Service.
- **Docitha Cardapio Digital**: cardapio digital com painel administrativo, login e CRUD, publicado no Azure Static Web Apps.
- **Portfolio IAS**: vitrine de videos e efeitos com IA para criadores de conteudo, publicado na Vercel.
- **Auto IA Videos**: automacoes em Python e FFmpeg para processamento e composicao de videos.

## Funcionalidades

- Home interativa com mapa holografico.
- Secao de projetos com videos demonstrativos.
- Links para GitHub e versoes online dos projetos.
- Curriculo disponivel para visualizar e baixar.
- Cards de certificados com links para PDFs.
- Layout responsivo para desktop e mobile.
- Fundo animado otimizado para reduzir travamentos.
